\section*{What the obstruction was}
The earlier code path forced Rockafellar potentials to be globally real-valued too early, which
introduced an artificial boundedness bottleneck in the Brenier pipeline.

\section*{What has been changed}
The formalization now uses a proper-convex potential
\[
  \Phi : E \to \mathbb R \cup \{+\infty\}
\]
(implemented as \texttt{WithTop~\(\mathbb R\)}), and runs Rockafellar containment at that level.

Concretely:
\begin{itemize}
\item proper subgradient objects are defined in
  \texttt{OptimalTransport/ProperConvex.lean};
\item Rockafellar is implemented via
  \texttt{properRockafellarPotential} in \texttt{OptimalTransport/Rockafellar.lean};
\item measurability and lower-semicontinuity of this potential are proved;
\item cyclical monotonicity yields the finiteness needed on the support-side locus;
\item support inclusion in \(\operatorname{Graph}(\partial \Phi)\) is proved globally in the
  proper-convex sense.
\end{itemize}

\section*{Current Brenier chain}
For quadratic cost \(c_\kappa(x,y)=\kappa\|x-y\|^2\), \(\kappa>0\), the code now proves:
\[
\text{optimality}
\Rightarrow
\text{pairing cyclical monotonicity of }\operatorname{supp}\pi
\Rightarrow
\text{proper Rockafellar containment}
\Rightarrow
\pi=(\mathrm{id},\nabla(\Phi^\ast))_{\#}\mu,
\quad
(\nabla(\Phi^\ast))_{\#}\mu=\nu,
\]
under the usual absolute continuity assumption \(\mu\ll m\), with no extra boundedness
predicate in the theorem assumptions.

\section*{Main theorem location}
The assembled endpoint is:
\[
  \texttt{Coupling.exists\_properConvex\_potential\_of\_optimal\_quadratic}
\]
in \texttt{OptimalTransport/Brenier.lean}, with Euclidean specialization also provided.

\section*{Status}
The mathematical obstruction documented in the earlier note is resolved in the current codebase.
